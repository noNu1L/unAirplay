===========================================
  unAirplay - Version Log / 版本日志
===========================================

This version log is intended for regular users.
Technical implementation details are not included.

此版本日志面向普通用户阅读，不包含技术实现细节。

---

v1.1.3
-------------------

1. FFmpeg Startup Check / FFmpeg 启动检查

   - Check FFmpeg installation before startup, show installation guide if not found
   - 启动前检查 FFmpeg 安装状态，未安装时显示安装指引

2. Docker Image Optimization / Docker 镜像优化

   - Reduced Docker image size
   - 减小 Docker 镜像体积


v1.1.2 (2026-02-07)
-------------------

1. AirPlay Device Offline Detection / AirPlay 设备离线检测

   - Devices offline for consecutive scans will be automatically removed
   - Configurations automatically restored when device comes back online
   - 设备连续多次扫描未检测到将自动删除虚拟设备
   - 设备重新上线时自动恢复配置

2. Bug Fix: Device Reconnection / 修复：设备重连问题

   - Fixed playback errors after device reconnects
   - 修复设备重连后的播放错误


v1.1.1 (2026-02-06)
-------------------

1. Streaming Audio Support / 流媒体音频支持

   - Added support for streaming audio sources (e.g., online radio)
   - 添加流媒体音频源支持（如网络电台）

2. AirPlay Device Exclusion / AirPlay 设备排除

   - Added AIRPLAY_EXCLUDE configuration to exclude devices by IP or name
   - 添加 AIRPLAY_EXCLUDE 配置，可按 IP 或名称排除设备

3. Bug Fix: Metadata Display / 修复：元数据显示

   - Fixed missing song metadata in some cases
   - 修复部分情况下歌曲元数据缺失的问题

4. Bug Fix: Multi-client Switching / 修复：多客户端切换

   - Fixed errors when switching between multiple DLNA clients
   - 修复在多个 DLNA 客户端之间切换时的错误


v1.1.0 (2026-02-04)
-------------------

1. Project Rename / 项目重命名

   - DLNA-To-Airplay -> unAirplay

2. Web UI Redesign / 页面重新设计

   - More minimalist design with integrated DSP settings
   - 更简约的设计，DSP 设置一体化

3. DSP Enhancement / DSP 增强

   - EQ & Spectral enhancement now support IIR/FFT/FIR modes
   - Spectral gain range expanded from 1.0-2.0 to 0.5-2.0
   - EQ 和频谱增强现在支持 IIR/FFT/FIR 三种模式
   - 频谱增强调整范围从 1.0-2.0 扩展至 0.5-2.0

4. Cache System / 缓存系统

   - Ensures continuous playback even when stream URL is interrupted
   - 保证音乐 URL 断流情况下仍能继续播放

5. Bug Fix / 修复

   - Fixed UI still showing "playing" status after track completion
   - 修复歌曲播放完成后页面仍显示播放中的问题

6. UI Enhancement / 界面优化

   - Playing devices now prioritized at the top of device list
   - 播放中的设备优先排列在设备列表顶部


v1.0.0 (Initial Release / 初始版本)
-----------------------------------

1. DLNA to AirPlay / DLNA 到 AirPlay

   - Stream audio from DLNA clients to AirPlay devices
   - 将 DLNA 客户端的音频推送到 AirPlay 设备

2. DLNA to Server Speaker / DLNA 到本地扬声器

   - Play audio on the server's local speaker
   - 在服务器本地扬声器播放音频

3. Music App Compatibility / 音乐应用兼容

   - Optimized DLNA requests for NetEase Cloud Music, QQ Music, Kugou Music, Kuwo Music, Migu Music
   - 针对网易云音乐、QQ音乐、酷狗音乐、酷我音乐、咪咕音乐的 DLNA 请求进行适配优化

4. DSP Features / DSP 功能

   - EQ, Spectral Enhancement, Dynamic Range Compression, Stereo Widening
   - EQ、频谱增强、动态范围压缩、立体声拓宽
